# SQMeeting iOS 客户端

# 系统要求

## 操作系统

最低支持 iOS13

# 编译生成和运行

## 编译工具

Xcode 15.0 或更高版本。
在build之前，请配置 Provisioning Profile 、Signing Certificate .

## 运行

配置好 服务地址、Provisioning Profile 、Signing Certificate ，直接 run 运行项目
